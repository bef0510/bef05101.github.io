DB 結構

  1. 檔案

    • 每個資料檔皆可設定檔案存放的位置，用以存放在不同的磁碟內，可改善效能，且減少Lock

  2. 檔案群組

    • 建立TABLE –>指定檔案群組 1

    • SQL Server會依照各檔案大小，依比例將資料表的資料平均寫入各資料檔中
    • 此TABLE的資料會分散在不同磁碟(假設次要資料檔分別位於不同磁碟)，提高效能

# 抓取網頁
import urllib.request as req
url = "https://www.ptt.cc/bbs/movie/index.html"
request = req.Request(url, headers={
    "User-Agent":"Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/75.0.3770.100 Safari/537.36"
})

with req.urlopen(request) as response:
    data = response.read().decode("utf-8")

# 解析原始碼
import bs4
root = bs4.BeautifulSoup(data, "html.parser")

# html 裡的 <title> 值 </title>
print(root.title) 
print(root.title.string)

# 尋找 class="title" 的 div 標籤
titles = root.find("div", class_="title") 
print(titles)
print(titles.a.string) # <a> 值 </a>

# 尋找所有 class="title" 的 div 標籤
titles = root.find_all("div", class_="title") 
print(titles) # 以列表方式找出 [<div> 值 </div>, <div> 值 </div>, <div> 值 </div>]

# 一筆一筆印出值
for title in titles:
    if title.a != None: # a 標籤沒有被刪除的資料
        print(title.a.string)

        self.x = x
        self.y = y

    def show(self):
        print(self.x, self.y)

    def distance(self, targetX, targetY):
        return(((self.x - targetX) ** 2) + ((self.y - targetY) ** 2)) ** 0.5

包裝檔案讀取的程式

建立兩個 txt 檔，任意輸入文字

File 實體物件的設計

class File:
    def __init__(self, name):
        self.name = name
        self.file = None # 尚末開啟檔案：初始值是 None

    def open(self):
        self.file = open(self.name, mode="r", encoding="utf-8")

    def read(self):
        return self.file.read()

# 讀取第一個檔案
f1 = File("data1.txt")
f1.open()
data = f1.read()
print(data)

# 讀取第二個檔案
f2 = File("data2.txt")

  1. 檔案配置

  2. 建立一個資料夾 geometry 及三個檔案 __init__.py 、line.py 、point.py 及一個主程式 main.py

  3. __init__.py 這個檔案一定要，讓 python 知道這個資料夾是一個封包，裡面不一定要有程式

line.py 寫入下方 code

# 計算兩個點的距離
def line(x1, y1, x2, y2):
    return (((x2 - x1) ** 2) + ((y2 - y1) ** 2)) ** 0.5

# 計算線段的斜率
def slope(x1, y1, x2, y2):
    return (y2 - y1) / (x2 - x1)

point.py 寫入下方 code

# 平面座標與原點距離
def distance(x, y):
    return (x ** 2 + y ** 2) ** 0.5

import geometry.point as point

# 計算兩個點的距離
def distance(x1, y1, x2, y2):

import geometry as geo
