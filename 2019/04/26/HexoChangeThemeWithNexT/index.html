Hexo 更換主題 NexT

發表於 2019-04-26 | 分類於 Blog

更換主題

安裝 NexT

使用 Git checkout

config 設定

根目錄 _config.yml 修改 theme

theme: next

Schemes 設定

themes/next/_config.yml 修改 Schemes

#Schemes
  scheme: Muse
  #scheme: Mist
  #scheme: Pisces
  #scheme: Gemini

menu 設定

將需要的打開

menu:
  home: / || home
  #about: /about/ || user
  tags: /tags/ || tags
  categories: /categories/ || th
  archives: /archives/ || archive
  #schedule: /schedule/ || calendar
  #sitemap: /sitemap.xml || sitemap
  #commonweal: /404/ || heartbeat

sidebar 設定

sidebar:
  position: left

新增 menu 對應資料

增加 tags，在 /source/ 下產生相對應資料

修改 title: 標籤
/source/tags/index.md 加入 type: "tags"

增加 categories，在 /source/ 下產生相對應資料

修改 title: 分類
/source/categories/index.md 加入 type: "categories"
