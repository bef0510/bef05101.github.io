安裝 Python

  1. 勾選 Add Python 3.7 to PATH → 點選 Install Now

  2. 點選 Disable path length limit

  3. 開啟 cmd → 輸入 python -v 查看版號

Visual Studio Code 安裝 Python Extension

  1. 開啟 VS Code

  2. 安裝 Python Extension

  3. 有未安裝的訊息，點選安裝

  4. 重啟 VS Code
