Python 資料型態

基本語法

變數，直接給值

age = 25
name = "Ross"
temperature = 36.2

單行注解

# age = 25
# name = "Ross"
# temperature = 36.2

多行注解

'''age = 25
name = "Ross"
temperature = 36.2'''

資料型態

Numbers 整數、長整數、浮點數(小數)

3456
3.45

String 任意的文字

"Hello World"

Boolean 布林值

True
False

List 可變列表，有順序、可變動的資料集合

[3, 4, 5]
["Hello", "World"]

Tuple 固定列表，有順序、不可變動的資料集合

(3, 4, 5)
("Hello", "World")

Set 集合，無順序的資料集合

{3, 4, 5}
{"Hello", "World"}

Dictionary 字典, key-value 的集合

{"name":"姓名", "age":"年齡"}
